# سایت آموزشی: هم‌نهشتی مثلث‌ها — پایه نهم

این پروژه یک سایت آموزشی کامل (فارسی) برای درس هم‌نهشتی مثلث‌ها است.
شامل:
- صفحهٔ اصلی (index.html)
- درسنامه تعاملی (darsnameh.html)
- تمرین‌ها و آزمون خودکار (tamrin.html)
- ابزار تعاملی SVG (interactive.html)
- استایل و اسکریپت‌های لازم (style.css, script.js)

## راه‌اندازی در GitHub Pages
1. یک مخزن (Repository) جدید ایجاد کنید (مثلاً ham-nehesti-site).
2. فایل‌های این پوشه را در ریشهٔ مخزن آپلود کنید.
3. در GitHub → Settings → Pages، شاخهٔ main و مسیر root را انتخاب کنید.
4. چند دقیقه صبر کنید؛ سایت با آدرس https://USERNAME.github.io/ham-nehesti-site فعال می‌شود.

## نکات معلمی
- بخش آزمون‌ها نمرهٔ خودکار دارد اما بخش نوشتاری باید توسط معلم بازبینی شود.
- می‌توانید SVGها را ویرایش کنید یا مثال‌های جدید اضافه کنید.
- برای ذخیرهٔ پاسخ‌ها به سرور نیاز است؛ در نسخهٔ فعلی ذخیرهٔ ساده با localStorage انجام می‌شود.
